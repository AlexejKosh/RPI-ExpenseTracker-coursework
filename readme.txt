Описание функциональности:
Приложение «Личный финансовый учёт» позволяет добавлять доходы и расходы с указанием суммы, даты, времени, категории и описания.
Все операции отображаются в журнале с возможностью удаления. Можно установить лимит бюджета на месяц и отслеживать его использование.
Для анализа расходов и доходов есть диаграммы по категориям с фильтром по дате и категории.
